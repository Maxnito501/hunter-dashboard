# 🐂 นายพรานจับจังหวะ (Hunter Dashboard)

แอปพลิเคชันวิเคราะห์ Bid/Offer 5 ช่อง ด้วย OCR + กลยุทธ์นายพราน  
พร้อมกุนซือทั้ง 5 ให้คำแนะนำ (ซุนวู, บัฟเฟต์, บิล เกตส์, อีลอน, เลโอนิดัส)

---

## ✨ ความสามารถหลัก

✅ อัปโหลดภาพหน้าจอ Bid/Offer  
✅ OCR อ่านตัวเลขจากภาพ  
✅ แก้ไขตัวเลขได้ก่อนวิเคราะห์  
✅ วิเคราะห์กลยุทธ์นายพราน 4 แบบ:
- 🐋 Whale Rider (ตามช้อน)
- 🎯 Reversal (จับจังหวะกลับตัว)
- 💀 Panic (หนีทันที)
- 🎣 Tired (รอซ้ำยามเปลี้ย)
✅ กุนซือ 5 ท่านให้คำแนะนำ  
✅ แสดงผลสวยงาม ใช้งานง่าย
